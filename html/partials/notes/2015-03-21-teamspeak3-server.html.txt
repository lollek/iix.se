<h2 id="test-hello-world">Test HELLO WORLD</h2>

<ol>
  <li>Download tarball from www.teamspeak.com</li>
  <li>Untar to /home/teamspeak</li>
  <li>Add user/group teamspeak</li>
  <li>Run the init as teamspeak &lt;pre&gt;
 sudo su - teamspeak
 ./ts3server_minimal_runscript.sh createinifile=1</li>
</ol>
<p>&lt;/pre&gt;
5. Configure ts3server.ini
6. Make sure you can start it (then ^C to kill it) &lt;pre&gt;
    ./ts3server_startscript.sh start
  &lt;/pre&gt;
7. Create /etc/systemd/system/teamspeak.service &lt;pre&gt;
    [Unit]
    Description=Teamspeak server
    After=network.target</p>

<div class="highlighter-rouge"><pre class="highlight"><code>[Service]
User=teamspeak
Group=teamspeak
Type=forking
PIDFile=/home/teamspeak/ts3server.pid
ExecStart=/home/teamspeak/ts3server_startscript.sh start

[Install]
WantedBy=multi-user.target
</code></pre>
</div>
<p>&lt;/pre&gt;
8. Test it thought systemctl and then enable it &lt;pre&gt;
    sudo systemctl start teamspeak.service
    sudo systemctl enable teamspeak.service
  &lt;/pre&gt;</p>
